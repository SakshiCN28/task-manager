# 🚀 Task Manager (MERN Stack)

A simple and clean Task Management Dashboard built using MERN Stack (MongoDB, Express, React, Node.js).

---

## 📌 Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 📋 View all tasks
- 🔄 Real-time updates

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB

---

## 📂 Project Structure

inten/

## ⚙️ Installation & Setup

### Clone repo

git clone https://github.com/SakshiCN28/task-manager.git

### Backend

cd backend  
npm install  
node server.js  

### Frontend

cd ../myapp  
npm install  
npm start
